# Sumario Executivo - Template QA Web

## Escopo
Automacao de QA para aplicacao web com:
1. login automatizado,
2. crawl exploratorio de menus,
3. coleta de evidencias,
4. relatorios para operacao e gestao.

## Diferenciais
1. Continue-on-error.
2. Re-login mid-crawl.
3. Multi-formato de saida (MD, JSON, CSV, Redmine).
4. Evidencias por pagina.

## Tempo estimado
1. Setup inicial: 10-20 min.
2. Crawl completo: depende do total de rotas e latencia do ambiente.

## Publico-alvo
1. QA analista
2. QA automacao
3. Engenharia de software
4. Lideranca tecnica
